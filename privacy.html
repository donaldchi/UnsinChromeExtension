Unsin Fortune Helper

비공식 Chrome 확장 프로그램 / 로컬 저장 전용

초안

1. 본 확장 프로그램에 대해

본 확장 프로그램은 unsin.co.kr 의 오늘의 운세 페이지에서 프로필 입력을 돕고, 사용자 본인의 브라우저 안에서 당일 결과를 임시로 보기 좋게 표시하는 비공식 도구입니다.

본 확장 프로그램은 운세의 신, 주식회사 스포츠조선, 기타 관련 회사가 제공하거나 승인하거나 제휴한 도구가 아닙니다.

2. 저장하는 정보

본 확장 프로그램은 폼 입력 보조를 위해 이름, 성별, 생년월일, 태어난 시간, 양력/음력 구분을 사용자 브라우저 안에 저장합니다. 저장에는 Chrome Extensions Storage API 의 local storage 를 사용합니다.

3. 저장하지 않는 정보

운세 결과는 데이터베이스, 외부 서버, Chrome의 영구 저장소에 저장하지 않습니다. 결과는 브라우저 세션 중 임시 표시 목적으로만 사용합니다.

본 확장 프로그램은 운세 결과의 번역, 요약, 판매, 재배포, 제3자 제공을 목적으로 하지 않습니다.

4. 외부 전송

입력된 프로필 정보는 사용자가 “오늘의 운세 보기”를 실행했을 때, unsin.co.kr 의 대상 폼에 입력하기 위한 목적으로 대상 사이트에 전송됩니다. 개발자의 서버나 외부 데이터베이스에는 전송되지 않습니다.

5. 제공 중단 및 변경

대상 사이트의 사양 변경, 이용약관 변경, 권리자의 요청, Chrome Web Store 정책상 문제, 또는 법령상 문제가 확인될 경우 본 확장 프로그램의 제공을 예고 없이 중단, 변경 또는 삭제할 수 있습니다.
