# Unsin Fortune Helper

`Unsin Fortune Helper`는 운세의 신(`unsin.co.kr`)의 오늘의 운세 결과를 Chrome 확장 프로그램 안에서 보기 좋게 정리해 보여주는 비공식 도우미입니다.

## 주요 기능

- 이름, 성별, 생년월일, 태어난 시간, 양력/음력 정보를 저장합니다.
- 저장한 프로필로 오늘, 어제, 내일의 운세를 불러옵니다.
- 운세 결과를 확장 프로그램 페이지 안에서 표시합니다.
- 특정 날짜를 선택해 해당 날짜의 운세를 확인할 수 있습니다.
- 표시 중인 날짜를 기준으로 전후 3일, 총 7일간의 `오늘의 운세 포인트`를 막대그래프로 보여줍니다.
- 결과를 `한마디 운세`, `애정운`, `소망운`, `사업운`, `금전운` 등 섹션별로 정리합니다.
- 원문 출처로 운세의 신 링크를 함께 표시합니다.

## 데이터 저장

- 프로필 정보: `chrome.storage.local`
- 최근 운세 결과와 7일 포인트: `chrome.storage.session`
- 별도 서버나 데이터베이스에는 저장하지 않습니다.

## 설치 방법

1. Chrome에서 `chrome://extensions/`를 엽니다.
2. 오른쪽 위의 개발자 모드를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 선택합니다.
4. 이 프로젝트 폴더를 선택합니다.
5. 확장 프로그램 아이콘을 눌러 프로필을 입력하고 저장합니다.
6. `오늘`, `어제`, `내일` 버튼으로 운세를 확인합니다.

## 사용 방법

1. 이름과 생년월일 정보를 입력합니다.
2. 성별, 태어난 시간, 양력/음력을 선택합니다.
3. `저장`을 누릅니다.
4. `오늘`, `어제`, `내일` 중 원하는 날짜를 선택합니다.
5. 결과 페이지에서 날짜 입력을 사용하면 특정 날짜의 운세도 확인할 수 있습니다.

## 파일 구성

- `manifest.json`: Chrome 확장 프로그램 설정
- `popup.html`, `popup.js`: 프로필 입력과 날짜 선택 화면
- `results.html`, `results.js`: 운세 결과와 7일 포인트 그래프 화면
- `background.js`: 운세 결과 요청, 파싱, 저장 처리
- `content.js`: 기존 본문 자동 입력/추출 로직
- `styles.css`: 공통 화면 스타일
- `privacy.html`: 개인정보 처리 안내

## 참고

이 확장 프로그램은 운세의 신 공식 서비스가 아닙니다. 운세 결과의 원문과 저작권은 운세의 신 및 해당 제공사에 있습니다.
